# v0.1.6 Release Showcase

This folder contains release evidence for the updated presentation skill.

The showcase compares the same synthetic deck topics three ways:

- native bundled Codex PPTX generation
- the published GitHub v1 baseline
- the updated skill path used for this release

Checked-in artifacts:

- `RELEASE_NOTES_v1.1.md`: release notes
- `release_showcase_manifest.json`: reproducible manifest and audit metrics
- `comparison-gallery/assets/comparisons/*.png`: 13 before/after contact sheets
- `comparison-gallery/build/presentation-skill-v1-1-release-gallery.pptx`: editable gallery deck

The full 39-deck matrix is generated by:

```bash
python3 scripts/build_release_showcase.py --clean
```

The generated per-case workspaces and intermediate build outputs are not meant
to be hand-edited. Rebuild them from the script when the release evidence needs
to be refreshed.
